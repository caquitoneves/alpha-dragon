# Alpha Dragon

Alpha Dragon é uma plataforma de engenharia inteligente capaz de transformar especificações técnicas em modelos paramétricos universais independentes de qualquer software CAD.

Nosso objetivo não é desenvolver apenas um plugin para AutoCAD.

Estamos construindo um Engineering Platform baseada em Inteligência Artificial, Engenharia Paramétrica e um Universal Engineering Model (UEM).

A arquitetura foi projetada para suportar diversos softwares CAD, múltiplos motores de IA e novos módulos de engenharia sem necessidade de reescrita do núcleo.

## Visão

Ser a principal plataforma mundial para automação do desenvolvimento de projetos mecânicos e industriais.
