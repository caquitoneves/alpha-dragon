# Domain Glossary

## Core Terms

- Engineering Project: the top-level container for a design effort.
- Assembly: a collection of parts and constraints forming a higher-level structure.
- Part: an engineering component with shape, material and manufacturing intent.
- Body: a geometric solid or shape inside a part.
- Feature: a modeled operation that changes geometry.
- Sketch: a parametric 2D construction used as the basis for features.
- Parameter: a named value controlled by expressions or constraints.
- Constraint: a logical or geometric rule that maintains consistency.
- Validation: the process of checking whether the model satisfies rules.
- Adapter: a connector to an external CAD or export system.
