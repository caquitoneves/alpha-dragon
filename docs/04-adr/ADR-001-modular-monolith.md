# ADR-001

## Título

Arquitetura inicial baseada em Modular Monolith.

## Status

Accepted

## Contexto

O MVP será desenvolvido por uma equipe pequena.

Microsserviços adicionariam complexidade desnecessária.

## Decisão

Adotar Modular Monolith.

Cada módulo possuirá fronteiras bem definidas.

## Consequências

Baixa complexidade.

Alta produtividade.

Migração futura para microsserviços quando houver necessidade.
