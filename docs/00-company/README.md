# Company Context

## Mission

Enable a new generation of engineering platforms that transform intent into reliable, parametric and interoperable engineering models.

## Vision

Alpha Dragon should become a global platform for intelligent engineering automation across product design, manufacturing planning and CAD interoperability.

## Strategic Intent

The project is not a simple CAD plugin. It is a long-term platform for engineering knowledge, parametric modeling, adapters, simulations and AI-assisted workflows.
