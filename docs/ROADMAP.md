# Platform Roadmap

## Phase 1 — Foundation

- consolidate architecture and domain concepts,
- establish documentation and principles,
- scaffold backend domain modules,
- define initial ADRs.

## Phase 2 — MVP Core

- implement the core project, part, assembly and feature model,
- implement parameter and constraint engine,
- add validation and history records.

## Phase 3 — Platform Expansion

- add CAD adapters,
- introduce plugin SDK,
- add export pipelines,
- enable knowledge and simulation layers.

## Phase 4 — Intelligence and Collaboration

- connect intent engine and AI assistants,
- expand collaboration features,
- support public API and SDK.
